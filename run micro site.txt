## 5️⃣ COMMON ISSUES & FIXES (VERY IMPORTANT)

### ❌ “command not found: npm”

You need Node.js.

Fix:

```bash
node -v
```

If not installed:

* Install **Node LTS** from nodejs.org
* Restart Terminal
* Re-run steps

---

### ❌ Port already in use

If 3000 is busy, the app will auto-pick another port.

Just open the URL it prints.

---

### ❌ npm install errors

Run:

```bash
rm -rf node_modules package-lock.json
npm install
```
